# My_Project
This is my first Project (Employee Management System) :-
Developed a console-based Employee Management System using core Java and Object-Oriented Programming (OOP) concepts. The application allows users to perform operations such as adding, removing, and displaying employee records through a menu-driven interface.
<br>
<br>
Implemented multiple classes including Employee, EmployeeManagementSystem, and the main class to ensure modular design. Utilized ArrayList from the Java Collections Framework to efficiently manage employee data, leveraging built-in methods like add() and remove() to simplify operations. User input was handled using the Scanner class.
<br>
<br>
The Employee class encapsulates employee details such as ID, age, and salary with proper getter and setter methods, and overrides the toString() method for formatted output. The system performs ID-based employee removal and uses iteration to display all records.
<br><br>
This project demonstrates a strong understanding of OOP principles, including encapsulation, class interaction, and dynamic data handling using collections
